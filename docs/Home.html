Project Description

This Utilty for adminstrating Windows services on local and remote machines

1. Start & Stop mulitple windows services.
2. Change password of the multiple windows services.

Some features of the tool

1. Multiple services can be chosen for Start or Stop. On click of Start or Stop button all the selected services will be Started or Stoped.

2. Similar way the user name and password can be applied to multiple servers.

3. When some text entered in Service name text box , Services containg the text in their display name will be listed.

4. Services with various status will be indicated using a color coding.

This has been developed using C# Windows Application using WMI.

Hope this tool is usefull and Any more suggestions to improve this tool is most welcome.


Note : Please make sure to run this project or the EXE as adminstrator.
